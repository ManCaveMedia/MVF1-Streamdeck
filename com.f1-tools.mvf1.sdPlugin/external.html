What do these values mean?

These values are used to configure the Stream Deck's connection to the MVF1 API.

The base URL is the URL of the MVF1 API. Find this by scrolling all the way to the bottom of the settings page of MVF1. It defaults to http://localhost:10101/api/graphql/

The timeout is the number of milliseconds to wait before timing out. If the MVF1 API is not responding, the plugin will wait this long before giving up. This defaults to 300ms.
